# Statistician-R
가짜연구소 3기 DSF 프로그램에서 Datacamp Statistician - R 강좌를 듣고 실습한 R 코드 답안 자료입니다.

ggplot, moderndive 등의 패키지를 비롯해 실습에 사용되는 데이터 셋 등은 Datacamp 실습 화면에선 이미 주어진 상태이기에
이 레포지토리에는 따로 데이터 파일과 라이브러리 import code는 commit하지 않았습니다.
